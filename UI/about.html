ABOUT

Pantasa is an innovative project aimed at enhancing the accuracy and efficiency of grammatical error correction in the Tagalog language. Using the rule-based techniques with deep learning methods, Pantasa a grammar checker will utilize language tools such as POS taggers, lemmatizers, and n-gram models to create pattern grammar rules as a basis for grammar error detection.

Pantasa contributes significantly to the field of Natural Language Processing (NLP) for low-resource languages, specifically Tagalog. Our key contributions include:

Enhanced Detection

Educational Impact

Enhanced Correction

Greater Coverage
